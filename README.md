# Trabajo Práctico - Programación III

## ¿Para qué sirve package.json?

El archivo package.json contiene la información del proyecto, las dependencias que utiliza y los comandos que pueden ejecutarse mediante npm.

## ¿Qué diferencia hay entre ejecutar Node y React?

Cuando ejecutamos un archivo con Node.js, el código JavaScript se ejecuta directamente en la consola.

En cambio, React utiliza un servidor de desarrollo (Vite) que compila la aplicación, permite trabajar con componentes y actualiza automáticamente el navegador cada vez que se modifica el código.
